# Cloud Image Surveillance

ESP32-CAM based cloud image surveillance system using Firebase.

## Overview

This project implements a motion-triggered image surveillance system using the ESP32-CAM and PIR sensor.

When motion is detected:
- The ESP32-CAM captures an image
- The image is compressed
- The image is uploaded to Firebase cloud storage using WiFi

The system demonstrates multimedia communication using embedded hardware and cloud infrastructure.

---

## Features

- Motion detection using PIR sensor
- ESP32-CAM image capture
- WiFi connectivity
- Firebase cloud upload
- Remote image storage
- Image resolution analysis
